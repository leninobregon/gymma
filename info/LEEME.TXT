============================================================
       SISTEMA DE GESTION DE GIMNASIO (GYM MA DB)
============================================================

DESCRIPCIÓN:
Este sistema permite administrar un gimnasio de forma eficiente,
controlando usuarios, accesos y cierres de caja con soporte
para multimoneda (Córdobas y Dólares).

------------------------------------------------------------
1. CARACTERÍSTICAS PRINCIPALES
------------------------------------------------------------
* GESTIÓN DE USUARIOS: Registro, edición y eliminación de 
  empleados (ADMIN y CAJEROS).
* CONTROL DE CAJA: Historial detallado de aperturas y cierres
  con cálculo automático de diferencias.
* TASA DE CAMBIO: Visualización destacada en tiempo real 
  para cobros exactos en USD/C$.
* SEGURIDAD: Encriptación de claves y protección de rutas 
  por niveles de acceso.
* REPORTES: Exportación de datos a Excel.

------------------------------------------------------------
2. ESTRUCTURA DE ARCHIVOS CRÍTICOS
------------------------------------------------------------
/classes/Usuario.php         -> Lógica de base de datos.
/controllers/UsuarioController.php -> Procesa las acciones.
/views/admin/gestion_usuarios.php  -> Interfaz visual.
/config/Database.php         -> Conexión al servidor MySQL.

------------------------------------------------------------
3. INSTRUCCIONES DE INSTALACIÓN
------------------------------------------------------------
1. Copiar la carpeta del proyecto en 'C:/xampp/htdocs/'.
2. Crear la base de datos en PHPMyAdmin (Ej: gym_db).
3. Configurar el archivo 'config/Database.php' con tu:
   - Usuario (root)
   - Contraseña
   - Nombre de la base de datos.
4. Acceder desde el navegador a: http://localhost/proyecto/

------------------------------------------------------------
4. NOTAS PARA EL ADMINISTRADOR
------------------------------------------------------------
* CAMBIO DE CONTRASEÑA: Al editar un usuario, si deja el 
  campo de contraseña vacío, el sistema conservará la clave
  que el usuario ya tenía.
* SEGURIDAD: El sistema NO permite que el administrador 
  que tiene la sesión iniciada se elimine a sí mismo.
* TASA DE CAMBIO: El diseño resalta el tipo de cambio en 
  color AMARILLO para evitar errores humanos al cobrar.
